# Simple.css demo

Shows poll-log-console on a [Simple.css](https://simplecss.org/) page (screenshot-friendly).

```bash
cd examples/simple-css
PYTHONPATH=../../src/python ../../.venv/bin/python app.py
# http://127.0.0.1:8774/
```

Expand the console, click **Ping** / **Local toast**, then capture the viewport.
