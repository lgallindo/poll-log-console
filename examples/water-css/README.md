# Water.css demo

Shows poll-log-console on a [Water.css](https://watercss.kognise.dev/) page (screenshot-friendly).

```bash
cd examples/water-css
PYTHONPATH=../../src/python ../../.venv/bin/python app.py
# http://127.0.0.1:8775/
```

Expand the console, submit a note or error toast, then capture the viewport.
